Opdracht 1

Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

  • Maak een externe stylesheet en koppel deze aan je HTML document.

    • Je hebt dan een apart bestand waarin je de stylesheet bijwerkt.
    Dit heeft een groot voordeel dat je veranderingen over de hele website door kunt voeren.
    Hierdoor zal de opmaak consistenter zijn/blijven.

  • Maak een stylesheet in het document zelf: dit is dan een interne stylesheet.

    • Bovenaan de website kun je ook de style sheet plaatsen. Voordeel is dat alles in een document zit.
    Nadeel is dat je bestand erg groot wordt doordat eerst alle opmaak geschreven moet worden.

  • Verander de stijl in een enkele HTML-tag.

    • Voor kleine websites is dit een prima oplossing. Je zet de opmaak daar waar je hem wil hebben.
    Maar zodra de website groeit wordt het onoverzichtelijk wat er nu precies waar gebeurd wat betreft styling.
